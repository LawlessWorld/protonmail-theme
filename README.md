## Installation

1 - Copy the CSS from /dist/css/prontmail.com
https://raw.githubusercontent.com/jeremiecook/protonmail-theme/master/dist/css/protonmail.css

2 - Paste it in ProntMail under Settings > Appearance tab.

You're done !
